# MusicNote Haven

**MusicNote Haven** creates practical desktop tools for musicians, MIDI collectors, keyboard enthusiasts and digital music archivists.

Our first application is **MusicNote Haven MIDI Manager**: a desktop application designed to help manage, organize, browse, play and preserve large MIDI and karaoke file collections.

## MusicNote Haven MIDI Manager

MIDI Manager is being built for people who work with large `.mid`, `.midi` and `.kar` archives and want a safer, clearer workflow for:

- indexing large MIDI/KAR collections
- searching and browsing archived files
- reviewing unknown or unsorted files
- organizing files into a cleaner library
- playing and checking MIDI/KAR files
- keeping source archives safe while building an Organized Library
- using Trial/Free, Personal and Professional workflows

The application is currently in active private development. Public downloads are not available yet.
